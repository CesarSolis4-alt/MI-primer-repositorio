# Propuesta de Mantenimiento Adaptativo  
## Proyecto: Calculadora de Presupuesto Mensual

### 1. Introducción
El mantenimiento adaptativo permite ajustar un software a los cambios del entorno tecnológico, garantizando que siga funcionando correctamente ante nuevas plataformas, dispositivos, navegadores o actualizaciones del sistema.  
Esta propuesta se centra únicamente en acciones de **mantenimiento adaptativo** aplicadas a la Calculadora de Presupuesto Mensual.

---

### 2. Objetivo del mantenimiento adaptativo
Asegurar que la calculadora continúe operando de manera estable frente a:

- Nuevas versiones de navegadores web  
- Cambios en resoluciones de pantalla  
- Actualizaciones de sistemas operativos  
- Modificaciones en librerías o dependencias  
- Necesidades de compatibilidad con dispositivos móviles modernos  

---

### 3. Situación actual del sistema
Actualmente, la Calculadora de Presupuesto Mensual funciona, pero presenta limitaciones cuando se utiliza en:

- Dispositivos móviles con pantallas pequeñas  
- Navegadores actualizados recientemente  
- Sistemas operativos que exigen estructuras más modernas de interfaz  
- Ambientes donde el modo oscuro afecta los colores actuales  
- Versiones nuevas de HTML/CSS/JavaScript  

Por ello, se requiere una actualización que permita una mayor compatibilidad general.

---

### 4. Cambios adaptativos propuestos

#### **1. Compatibilidad con versiones nuevas de navegadores**
Ajustar el código para evitar que funciones antiguas de JavaScript fallen en versiones recientes de Chrome, Firefox, Edge o Safari.

#### **2. Diseño responsivo actualizado**
Rediseñar la interfaz para que se adapte mejor a pantallas de:

- Teléfonos  
- Tablets  
- Pantallas 4K  

Usando unidades flexibles (rem, %, vw, vh) y media queries optimizadas.

#### **3. Implementación de modo claro/oscuro**
Adaptar colores, botones y textos para que no haya problemas de visibilidad en modo oscuro.

#### **4. Actualización de librerías**
Actualizar librerías como:

- Frameworks de estilos  
- Iconos  
- Scripts auxiliares  

Evita incompatibilidades con versiones antiguas.

#### **5. Optimización para sistemas operativos recientes**
Ajustes para mejor funcionamiento en:

- Android 14  
- Windows 11  
- iOS 17  

#### **6. Soporte para PWA (si se desea en futuro)**
Preparar la base para que la aplicación pueda instalarse como app ligera en celulares.

---

### 5. Beneficios esperados

- Funcionamiento correcto en dispositivos modernos  
- Mayor accesibilidad en pantallas pequeñas  
- Interfaz flexible y actual  
- Menos errores causados por incompatibilidades  
- Mejor desempeño en navegadores recientes  
- Preparación para futuras expansiones del software

### 7. Conclusión
El mantenimiento adaptativo garantizará que la Calculadora de Presupuesto Mensual continúe funcionando sin problemas ante cambios tecnológicos futuros.  
Al aplicar estas mejoras, el sistema será más estable, compatible y duradero, manteniendo su utilidad para los usuarios independientemente del dispositivo o plataforma donde se ejecute.
